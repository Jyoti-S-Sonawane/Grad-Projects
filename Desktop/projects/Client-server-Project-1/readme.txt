Steps to run Client-server-Project-1:

1. Unzip the "Client-server-Project-1" folder
2. Open two terminals with "Client-server-Project-1" folder path
3. In first terminal, use below command to compile the "server.c" file
	gcc server.c -o server
4. Then use below command to run the server file
	./server
5. In second terminal, use below command to compile the "client.c" file 
	gcc client.c -o client
6. Then use below command to run the client file
	./client
7. Once both client and server started running, use the client's terminal window to select the test cases for execution.
