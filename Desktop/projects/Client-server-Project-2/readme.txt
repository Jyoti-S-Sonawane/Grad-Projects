Steps to run Client-server-Project-2:

1. Unzip the "Client-server-Project-2" folder
2. Open two terminals with "Client-server-Project-2" folder path
3. In first terminal, use below command to compile the "server2.c" file
	gcc server2.c -o server2
4. Then use below command to run the server file
	./server2
5. In second terminal, use below command to compile the "client2.c" file 
	gcc client2.c -o client2
6. Then use below command to run the client file
	./client2
7. Once both client and server started running, use the client's terminal window to select the test cases for execution.
